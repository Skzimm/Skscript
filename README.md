# SKScript by Skzim

**SKScript** is a custom-made Roblox cheat developed by **Heitor (aka Skzim)**, designed for Counter Blox and similar titles. Built with versatility in mind, it offers multiple modes and features to suit different playstyles — from subtle enhancements to full-on rage mode.

---

## 📌 Description

SKScript is a Lua-based cheat for Roblox that provides advanced aiming assistance, ESP visuals, and automated gameplay tools. It’s designed to be flexible, allowing users to switch between legit and rage configurations depending on their needs.

---

## ⚙️ Features

- **Aimbot** – Automatically aims at targets with adjustable strength  
- **Aimlock** – Locks aim onto a specific target for consistent tracking  
- **ESP** – Displays players through walls and obstacles  
- **Charms** – Visual enhancements for player models  
  - **Team Check** – Filters ESP to show only enemies  
  - **Visibility Check** – Highlights only visible targets  
- **Wall Penetration Analyzer** – Detects if a surface can be shot through; in Rage mode, penetrable targets are marked with a blue box  
- **Autocrunch** – Automatically crouches during combat for better accuracy  
- **Autofire (Beta)** – Automatically fires when a target is detected  
- **Autostop (In Development)** – Stops movement before shooting to improve precision

---

## 🎮 Modes

SKScript includes 5 distinct modes to match your gameplay style:

| Mode        | Description |
|-------------|-------------|
| **Nada**        | All features disabled; default behavior |
| **Full Legit**  | Weak Aimbot with very low smoothness |
| **Legit**       | Weak Aimbot, low smoothness, ESP enabled |
| **Semi**        | Medium Aimbot, medium smoothness, ESP and Charms |
| **Rage**        | Strong Aimbot, Aimlock, zero smoothness, ESP, Charms, Autocrunch, Autofire (testing phase), Autostop (in progress) |
